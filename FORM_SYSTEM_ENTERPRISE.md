# FORM_SYSTEM_ENTERPRISE

## Sistema unificado

- Labels de alta legibilidad (`crm-label`).
- Grid de formulario reusable (`crm-form-grid`, `crm-col-*`).
- Estados de error consistentes (`crm-input-error`, `crm-error-text`).
- Focus ring accesible unificado.

## Compatibilidad
No requiere backend nuevo; se aplica sobre Razor forms actuales.
