# FRONTEND_GOVERNANCE_FINAL

## Reglas finales
1. `crm-*` obligatorio en nuevas implementaciones UI.
2. No inline styles en código nuevo.
3. Shared partials first para patrones repetibles.
4. Responsive-first en todo cambio visual.
5. Accessibility-first en interacción y foco.
6. Operational UX first para uso diario.
7. SaaS consistency first para experiencia comercial.
