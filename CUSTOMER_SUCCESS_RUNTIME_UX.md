# CUSTOMER_SUCCESS_RUNTIME_UX

## Continuidad adopción
- Onboarding dismissible por módulo (`localStorage`).
- Reset onboarding desde ops bar Index (`data-crm-onboarding-reset`).
- Quick start grid en dashboard.
- Barra runtime facilita educación “dónde está cada cosa”.

## Soporte operativo
Enlace Support en ops bar Index; módulo Support en sidebar con `ModuleActive`.

## CS en runtime
Usuarios nuevos: runtime bar como mapa mental persistente.
Usuarios expertos: barra no intrusiva, sticky solo desktop.

## Métricas adopción sugeridas
- % usuarios que ocultan onboarding vs completan quick start.
- Módulos más visitados vía runtime bar (instrumentación futura).
