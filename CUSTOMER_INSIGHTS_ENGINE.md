# CUSTOMER INSIGHTS ENGINE

`ICustomerInsightsEngine` → `CustomerInsightsEngine`

## Insights generados
- Cliente más / menos activo (engagement)
- Módulos ignorados vs críticos
- Stickiness bajo del producto
- Patrón de adopción agregado

Cada insight: tipo, severidad, `Actionable`, módulo/cliente opcional.

## API
`GET /api/intelligence/insights?tenantId=`
