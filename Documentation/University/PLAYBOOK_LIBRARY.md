# PLAYBOOK LIBRARY

> Procedimientos paso a paso — ejecutar en orden

## Cliente Nuevo

1. Lead calificado o referido
2. Crear/verificar Customer
3. Customer 360 completo
4. Deal onboarding si aplica
5. Handoff a CS con nota
6. Tareas día 1-7

```mermaid
flowchart TD
    S[Inicio]
    S --> P0[Lead calificado o referido]
    S --> P1[Crear/verificar Customer]
    P0 --> P1
    S --> P2[Customer 360 completo]
    P1 --> P2
    S --> P3[Deal onboarding si aplica]
    P2 --> P3
    S --> P4[Handoff a CS con nota]
    P3 --> P4
    S --> P5[Tareas día 1-7]
    P4 --> P5
```
