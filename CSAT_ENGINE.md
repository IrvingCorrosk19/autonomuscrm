# CSAT ENGINE

`ICsatEngine` → `CsatEngine`

## Escala
Encuestas **1–5** en `CustomerFeedbacks` tipo `CSAT`.

## Métricas
- Promedio global
- **Trend delta** — últimos 30d vs 30d anteriores
- Historial mensual (12 períodos)

## API
- `GET /api/intelligence/csat`
- `POST /api/intelligence/csat?customerId=&score=`
