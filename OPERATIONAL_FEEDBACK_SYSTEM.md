# OPERATIONAL_FEEDBACK_SYSTEM

- `crmUi.toast` soporta transición suave, acciones contextuales y retry UX.
- `crmUi.trackOperation` unifica feedback de operaciones async con jerarquía success/error.
- Alerts operacionales migradas a componentes de estilo consistente (`crm-alert-floating`).
- Feedback más legible para entornos multi-acción de operación SaaS.
